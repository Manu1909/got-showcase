Showcase App

GoT

Manuel Entenmann - 12.02.2020

Aufgabe

"Erstelle eine Single-Page-Application. Die SPA soll einem Benutzer alle Häuser aus Game of Thrones in einer Übersicht anzeigen."

Technologien

  • Angular als SPA Framework
  • Bootstrap als responsive web design library
  • Adobe Xd als Mockup Tool
  • Github als Sourcecodeverwaltung

Idee

  1. Darstellung aller Häuser in zwei Kategorien unterteilen
  2. Bei Klick auf diese Kategorie wird die entsprechende Liste der Häuser angezeigt
  3. Innerhalb der Liste kann ein einzelnes Haus ausgewählt werden.
  4. Die Anwendung leitet den Benutzer dann entsprechend auf eine Seite weiter, welche detaillierte Information über das Haus beinhaltet.
